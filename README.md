# Debounce Search App

A responsive React.js application that implements debounce functionality with live search using API integration.

## Live Demo 

https://debounce-search-omega.vercel.app/

## Features

- Real-time search functionality
- Debounce implementation for optimized API calls
- Dynamic search results display
- Responsive UI for mobile and desktop
- Loading and error handling
- Clean and user-friendly interface

## Tech Stack

- React.js
- JavaScript
- CSS3
- REST API

## Concepts Used

- React Hooks (`useState`, `useEffect`)
- Debouncing with `setTimeout`
- API Fetching
- Conditional Rendering
- State Management

## Usage
- Type in the search input
- API calls are delayed using debounce
- Search results appear dynamically
- Project Goal

## This project was built to understand:

- Performance optimization
- Reducing unnecessary API requests
- Handling dynamic search efficiently
